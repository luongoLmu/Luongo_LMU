Api's

An API (Application Programming Interface) is an interface that allows different
software applications to communicate with each other and exchange data.

APIs can be implemented in various programming languages and are an important part of modern software development

Tik Tok Endpoints

Tik Tok Login Request/Registration : Post ( Putting in Information)

Search Engine: Post (putting in Search ) , Get ( getting the searched videos )

View Video: Get (server is showing the videos)

Like , comment, share : Post (interacting with video )

Save Videos: Post ( press save button and putting them in a ) Get ( shows videos of followed account)

Create and Share Video: Post (creating video) , Get ( showing the video on the Platform)

Video Analytics: Get ( shows and generates Profile and Video analytics )

Re post: Post ( Using repost Button), Get ( shows what videos you and friends reposted )

User Profiles : Get ( Viewing own or other Profiles )

follow other Profiles: Post ( pressing follow Button) Get ( shows videos of followed account)

Messenger: Post (writing a Person) Get ( seeing the Message of the other Person)
